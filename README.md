**How To install This Tool** 

**STEP 1 : git clone https://github.com/n00bC0der69/WebUp.git**

**STEP 2 : cd WebUp**

**STEP 3 : cp webup /bin**

**STEP 4 : chmod +x webup**

**NOW YOU CAN USE WEBUP IN ANY DIR**

**How To Use This Tool**

webup **PORT NUM YOU WANT TO HOST**
